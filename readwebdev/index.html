Preamble

Books and articles that I've read or plan to read. Good stuff.

The Languages

HTML and CSS

Beginner

HTML - Wikipedia
  • An overview of HTML
Codecademy - HTML&CSS
  • Quite helpful if you’re learning-by-doing type of learner
  • It’s very basic; much more work is needed
Head First HTML and CSS
  • Gentle introduction with exercises.
  • Might be slow for people who with previous experiences.
  • Also it’s a rather old book. Fortunately HTML is mostly backward compatible.
Index of HTML5 Element
  • Quick reference of all HTML5 elements. It’s like the Specification, but better formatted and easier to read.
  • Read through to check for unknown unknowns.

Intermediate

CSS: The Definitive Guide
  • A very detailed description of the CSS syntax. No exercises.
  • Only introduces up to CSS 2.1.
  • 4th version partly available here

Advanced

The HTML Standard (WHATWG)
The HTML5.0 Standard (W3C)
The CSS Specification 2.1
The Current CSS Specifications
The XMLHttpRequest Specification

Supplementary

Dive into HTML5
  • Pleasant read with anecdotes and little humours
  • Moderately useful
  • Maintenance ended in early 2011, which is an ancient era for Web

JavaScript, DOM, and BOM

Beginner

JavaScript - Wikipedia
Codecademy - JavaScript
  • A very basic look into the essentials

Intermediate

JavaScript: The Good Parts
  • Nice little book about the key features of JavaScript.
  • Very opinionated, but seems reasonable.
Professional JavaScript for Web Developers
  • Comprehensive and detailed introduction to JavaScript the language and DOM and relevant technologies
  • Update in 2011, recent enough
ES6 Overview in 350 Bulletpoints
  • Useful for covering unknown unknowns

Advanced

ECMAScript 2015 Language Specification
DOM Specifications

JavaScript Quiz. ES6 edition.

Reference

Mozilla Developer Network
  • Very comprehensive reference HTML, CSS, JavaScript, and more, filled with examples

Interactive Data Visualization for the Web
  • Freely available online
  • Step-by-step introudction to D3

D3 Tips and Tricks
  • Freely available online
  • An edited compilation of a blog
  • Useful for more experienced D3 users

The Pixel Factory - WebGL, GPUs and Math(Box)

  • Vivid presentation
