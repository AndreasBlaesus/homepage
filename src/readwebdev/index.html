Webdev Reading List

Preamble

Books and articles on Web development (leaning toward frontend) that I find useful.

They're sorted roughly in order of my own learning.

The Languages

HTML and CSS

Beginner

World Wide Web (CERN)
  • The first website in history. Learn about the original vision.
  • Also, take a look at how HTML was written (and how they're rendered by modern browsers).
HTML - Wikipedia
  • An overview of HTML
Codecademy - HTML&CSS
  • Quite helpful if you’re learning-by-doing type of learner
  • It’s very basic; much more work is needed
Head First HTML and CSS
  • Gentle introduction with exercises.
  • Might be slow for people who with previous experiences.
  • Also it’s a rather old book. Fortunately HTML is mostly backward compatible.
Index of HTML5 Element
  • Quick reference of all HTML5 elements. It’s like the Specification, but better formatted and easier to read.
  • Read through to check for unknown unknowns.

Intermediate

CSS: The Definitive Guide
  • A very detailed description of the CSS syntax. No exercises.
  • Only introduces up to CSS 2.1.
  • 4th version partly available here
The Languages Which Almost Became CSS
  • History of CSS (and its competitors)
CSS Selector Level 4 (draft)
  • Very worthwhile to master CSS selectors
  • Level 4 is much clearer than level 3

Advanced

The HTML Standard (WHATWG)
The Current CSS Specifications

Supplementary

Dive into HTML5
  • Pleasant read with anecdotes and little humours
  • Moderately useful
  • Maintenance ended in early 2011, which is ancient in Web world

JavaScript, DOM, and BOM

Beginner

JavaScript - Wikipedia
  • Grand overview of the language
Codecademy - JavaScript
  • A very basic look into the essentials

Intermediate

JavaScript: The Good Parts
  • Nice little book about the key features of JavaScript.
  • Very opinionated, and some practices have become obsolete in modern development.
Professional JavaScript for Web Developers
  • Comprehensive and detailed introduction to JavaScript the language and DOM and relevant technologies
  • Update in 2011, quite old
ES6 Overview in 350 Bulletpoints
  • Useful for covering unknown unknowns

Advanced

ECMAScript 5.1 Language Specification
  • This is an old version, but it's also thinner to read.
TC39 Proposals
  • Keep an eye on new ES features
  • Also pay attention to the proposal process

Supplementary

You Don’t Know JS
  • The title might be too strong, but the books are helpful.
You might not need jQuery
  • Nice collection of frequent operations; useful even for those who don't use jQuery (like myself)

Reference (for HTML & CSS & JS)

Mozilla Developer Network
  • Very comprehensive reference for HTML, CSS, JavaScript, and more, with examples
Can I Use
  • Quickly check browser compatibility of a feature

Library, Framework, and API

Philosophy

Model View Controller: History, theory and usage
  • Nice introduction to difference patterns of MVC

React and Friends

React

React Official Documentation
  • Better read everything here. At least in September 2015, this is still feasible and worthwhile.
  • The reference section is a bit terse but definitely worth the time.

Flux

Flux Official Documentation

Relay

Relay Official Documentation

Visualisation

d3.js

Interactive Data Visualization for the Web
  • Freely available online
  • Step-by-step introudction to D3
  • Published in 2013 but most contents are relevant
  • Despite the name, the book really only talks briefly about the “Holy Trinity” of Web Development and then focuses on D3; not much on “Data Visualization” in general or as a disciplline
D3 Tips and Tricks
  • Freely available online
  • An edited compilation of a blog
  • Useful for more experienced D3 users

WebGL, three.js

The Pixel Factory - WebGL, GPUs and Math(Box)
  • Basic introduction to computer Graphics in 3D
  • Vivid presentation
